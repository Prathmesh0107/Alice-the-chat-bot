# Alice - AI Chat Bot 🤖

Alice is an intelligent AI-powered chatbot built using Python that enables users to interact in natural language conversations. The chatbot is designed to provide quick, contextual, and human-like responses through an easy-to-use interface.

## 🚀 Features

* Natural language conversation with AI
* Real-time user interaction
* Clean and simple interface
* Fast response generation
* Easy to customize and extend
* Beginner-friendly project structure

## 🛠 Tech Stack

* **Python**
* **Streamlit** (Frontend / UI)
* **LLM / AI Model Integration**
* **Virtual Environment**
* **Git & GitHub**

```bash
Alice-the-chat-bot/
```

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Prathmesh0107/Alice-the-chat-bot.git
cd Alice-the-chat-bot
```

### 2. Create virtual environment

```bash
python -m venv venv
```

### 3. Activate environment

**Windows**

```bash
venv\Scripts\activate
```

**Mac/Linux**

```bash
source venv/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

## ▶️ Run the Project

```bash
python chatbot.py
```

Or if built with Streamlit:

```bash
streamlit run chatbot.py
```
